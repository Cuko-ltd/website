# Cuko Ltd — Website

Marketing and content site for **Cuko Ltd**, Samuel Ventimiglia's fractional Chief Technology Officer consultancy. London-based; sectors: crypto-native finance, healthcare, AI/agent infrastructure, cloud-native.

Live at **[https://cuko.uk](https://cuko.uk)**. Built with Astro 5, hosted on Cloudflare Pages.

## Developer documentation

Stack, repository layout, build scripts, and the editorial/publishing workflow are documented in [`CLAUDE.md`](./CLAUDE.md).

Quick start:

```bash
npm install
npm run dev      # local dev server
npm run build    # prebuild OG image, then astro build
npm test         # Playwright suite (smoke / a11y / visual)
```
